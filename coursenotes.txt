[Dont compare float variables directly]
the value stored internally for a float object may not be precisely what youâ€™d think it would be. For that reason, it is poor practice to compare floating-point values for exact equality. Consider this example:

>>> x = 1.1 + 2.2
>>> x == 3.3
x=1.1+2.2
print (x)
3.3000000000000003

Hence better way of doing this is 
tolerance = 0.00001
>>> x = 1.1 + 2.2
>>> abs(x - 3.3) < tolerance
True
